Introduction to PHP

PHP, which stands for "PHP: Hypertext Preprocessor," is a widely-used open-source server-side scripting language designed for web development. It is embedded within HTML, allowing developers to create dynamic web pages that interact with databases and user inputs.

Key Features of PHP

  • Ease of Use: PHP is known for its simplicity, making it easy for beginners to learn and for experienced developers to quickly build complex applications.
  • Cross-Platform: PHP runs on various platforms, including Windows, Linux, and macOS, allowing developers to work in diverse environments.
  • Database Integration: PHP supports various databases, including MySQL, PostgreSQL, and SQLite, enabling developers to create data-driven applications effortlessly.
  • Strong Community Support: With a vast community of developers, PHP offers extensive documentation, tutorials, and frameworks, ensuring help is readily available.
  • Extensive Functionality: PHP includes a rich set of built-in functions and libraries for tasks such as file handling, session management, and data validation.
  • Frameworks and CMS: Popular frameworks like Laravel, Symfony, and CodeIgniter, along with content management systems like WordPress and Drupal, are built on PHP, providing robust tools for web development.

Common Use Cases

PHP is commonly used for:

  • Developing dynamic web applications and websites.
  • Building e-commerce platforms and content management systems.
  • Creating APIs and web services to facilitate data exchange between applications.
  • Handling form submissions and user authentication for web applications.
  • Interacting with databases to store, retrieve, and manipulate data.

Getting Started with PHP

To begin programming in PHP, you need a server environment, such as XAMPP or WAMP, which includes a PHP interpreter and a web server (Apache). You can write PHP scripts in any text editor and save them with a .php extension. When accessed via a web browser, the PHP interpreter executes the code on the server and returns the resulting HTML to the client.

Installing PHP

To develop PHP applications, you need a server environment that includes PHP, a web server, and a database. Two popular options for setting up this environment are WAMP and XAMPP.

Installing WAMP

WAMP (Windows, Apache, MySQL, PHP) is a Windows-based web development environment that allows you to create web applications with Apache2, PHP, and MySQL.

Steps to Install WAMP:

  1. Download WAMP: Go to the official WAMP server website: WAMP Server and download the latest version.
  2. Run the Installer: Double-click the downloaded file and follow the installation prompts. Choose your preferred language and install it in the desired location.
  3. Start WAMP: Once installed, launch WAMP. You’ll see a WAMP icon in your system tray.
  4. Check Server Status: The WAMP icon will turn green if the server is running properly. If it’s orange or red, there may be issues with Apache or MySQL.
  5. Accessing WAMP: Open a web browser and type http://localhost to access the WAMP dashboard, where you can manage databases and PHP settings.

Installing XAMPP

XAMPP is a cross-platform web server solution stack package that includes Apache, MySQL, PHP, and Perl. It is available for Windows, macOS, and Linux.

Steps to Install XAMPP:

  1. Download XAMPP: Go to the official XAMPP website: Apache Friends and download the version suitable for your operating system.
  2. Run the Installer: Double-click the downloaded file and follow the installation wizard. Choose the components you want to install (ensure PHP and MySQL are selected).
  3. Start XAMPP Control Panel: After installation, open the XAMPP Control Panel. You’ll see a list of modules like Apache and MySQL.
  4. Start Services: Click the “Start” buttons next to Apache and MySQL to run them. The background of the module should turn green.
  5. Accessing XAMPP: Open a web browser and type http://localhost to access the XAMPP dashboard.

Testing Your Installation

To confirm that PHP is installed correctly, create a simple PHP file:

            <?php
            phpinfo();
            ?>

Save this as info.php in the www directory of WAMP or the htdocs directory of XAMPP. Then navigate to http://localhost/info.php in your web browser. You should see a page displaying your PHP configuration details.

PHP Syntax

PHP syntax consists of a series of rules that govern how PHP code is written and structured. Understanding PHP syntax is essential for effective programming.

Basic Syntax

PHP code is typically embedded within HTML documents. A PHP script begins with the <?php opening tag and ends with the ?> closing tag:

            <?php
            // Your PHP code goes here
            echo "Hello, World!";
            ?>

Embedding PHP Code in a Web Page

To embed PHP code in an HTML page, simply include the PHP tags within the HTML code. The PHP interpreter processes the PHP code on the server, and the output is sent to the client as HTML:

            <html>
            <head>
                <title>My PHP Page</title>
            </head>
            <body>
                <h1>Welcome to My PHP Page</h1>
                <?php
                    echo "<p>This is a paragraph generated by PHP.</p>";
                ?>
            </body>
            </html>

In this example, the PHP code within the body generates a paragraph that will be displayed on the web page.

Outputting Data

To output data to the web page, you can use the echo or print statements. Both can display strings, numbers, and variables:

            <?php
            $name = "Alice";
            echo "Hello, " . $name . "!"; // Outputs: Hello, Alice!
            ?>

Using PHP in HTML Attributes

You can also embed PHP code within HTML attributes. This is useful for dynamically generating values:

            <?php
            $imageSrc = "image.jpg";
            ?>
            <img src="<?php echo $imageSrc; ?>" alt="My Image">

Short Echo Tags

For convenience, PHP provides short echo tags, which can be used to output values without needing to write echo explicitly:

            <?php
            $age = 25;
            echo "I am {$age} years old."; // Outputs: I am 25 years old.
            ?>

Working with Variables in PHP

Variables in PHP are used to store data, which can be reused throughout your scripts. PHP variables are flexible and do not require explicit declaration of data types.

Variable Naming Rules

  • Variable names must start with a letter or an underscore (_).
  • Subsequent characters can include letters, numbers, and underscores.
  • Variable names are case-sensitive.
  • Variables cannot be PHP reserved keywords.

Token

In PHP, a token is a basic unit of the language's syntax. Tokens are the smallest elements in a script, representing the individual components that make up PHP code. The types of tokens in PHP include:

  • Variables: Identified by a dollar sign ($) followed by the variable name (e.g., $variableName).
  • Operators: Symbols that represent operations (e.g., + for addition, - for subtraction).
  • Keywords: Reserved words in PHP that have special meaning (e.g., if, else, function).
  • Strings: Text data enclosed in quotes (e.g., "Hello, World!").
  • Numbers: Numeric values (e.g., 123, 3.14).
  • Symbols: Special characters that serve specific purposes (e.g., parentheses ( ), braces { }).

Declaring Variables

To declare a variable in PHP, simply prefix the variable name with a dollar sign ($):

            <?php
            $greeting = "Hello, World!";
            echo $greeting; // Outputs: Hello, World!
            ?>

Variable Types

PHP supports various data types for variables:

  • String: A sequence of characters.
  • Integer: A whole number.
  • Float: A decimal number.
  • Boolean: Represents true or false.
  • Array: A collection of values.
  • Object: An instance of a class.
  • NULL: Represents a variable with no value.

Variable Scope

Variable scope defines the context in which a variable is accessible. PHP has four types of variable scope:

  • Local: Variables defined within a function are accessible only inside that function.
  • Global: Variables defined outside of functions can be accessed using the global keyword inside a function.
  • Static: Variables declared with the static keyword retain their value across function calls.
  • Parameter: Variables passed to functions as arguments.

Functions in PHP

Functions are blocks of code that can be reused in a program. They help organize your code into manageable pieces and reduce redundancy.

Function Definition

A function is defined using the function keyword, followed by a name and a set of parentheses:

            <?php
            function greet() {
                return "Hello, World!";
            }
            ?>

Creating and Invoking Functions

You can create a function and then invoke (call) it:

            <?php
            function greet() {
                return "Hello, World!";
            }
            echo greet(); // Outputs: Hello, World!
            ?>

User-defined Functions

You can create your own functions to perform specific tasks:

            <?php
            function add($a, $b) {
                return $a + $b;
            }
            echo add(5, 10); // Outputs: 15
            ?>

Formal Parameters and Actual Parameters

Formal parameters are the variables defined by the function; actual parameters are the values you pass: 

            <?php
            function multiply($x, $y) { // $x and $y are formal parameters
                return $x * $y;
            }
            echo multiply(4, 5); // Outputs: 20 (4 and 5 are actual parameters)
            ?>

Function and Variable Scope

The scope of a variable defines where it can be accessed:

            <?php
            $globalVar = "I am a global variable";
    
            function testScope() {
                global $globalVar; // Accessing the global variable
                $localVar = "I am a local variable";
                echo $globalVar . "
"; // Outputs: I am a global variable
                // echo $localVar; // Uncommenting this will cause an error
            }
            testScope();
            ?>

Recursion

A function can call itself, which is known as recursion:

            <?php
            function factorial($n) {
                if ($n <= 1) {
                    return 1;
                }
                return $n * factorial($n - 1);
            }
            echo factorial(5); // Outputs: 120
            ?>

Library Functions

PHP provides many built-in library functions:

Example: Array Functions

            <?php
            $array = [1, 2, 3, 4, 5];
            $sum = array_sum($array); // Sum of array elements
            echo $sum; // Outputs: 15
            ?>

Example: String Functions

            <?php
            $string = "Hello, World!";
            $length = strlen($string); // Length of the string
            echo $length; // Outputs: 13
            ?>

Strings in PHP

Strings are a sequence of characters, and PHP provides many functions to manipulate them.

Defining Strings

            <?php
            $string1 = "Hello";
            $string2 = 'World';
            ?>

String Interpolation

You can embed variables inside double-quoted strings using curly braces:

            <?php
            $name = "John";
            echo "Hello, {$name}!"; // Outputs: Hello, John!
            ?>

Heredoc Syntax

Heredoc is a way to define strings over multiple lines:

            <?php
            $heredoc = <<<EOD
            This is a heredoc string.
            It can span multiple lines.
            EOD;
            echo $heredoc;
            ?>

String Functions

Common string functions include:

  • strlen(): Get the length of a string.
  • strtoupper(): Convert to uppercase.
  • strtolower(): Convert to lowercase.
  • substr(): Extract a substring.
  • strpos(): Find the position of the first occurrence of a substring.

Formatting Strings

You can format strings using the printf() function:

            <?php
            $name = "John";
            $age = 25;
            printf("My name is %s and I am %d years old.", $name, $age);
            ?>

Comparing and Searching Strings

You can compare strings using comparison operators:

            <?php
            $str1 = "Hello";
            $str2 = "hello";
    
            if ($str1 === $str2) {
                echo "Strings are identical.";
            } else {
                echo "Strings are different.";
            }
            ?>

To search for a substring:

            <?php
            $haystack = "Hello, welcome to my world.";
            $needle = "welcome";
    
            if (strpos($haystack, $needle) !== false) {
                echo "Found '$needle' in the string.";
            } else {
                echo "'$needle' not found.";
            }
            ?>

Character and String Indexing

You can access individual characters in a string using indexing:

            <?php
            $str = "Hello";
            echo $str[0]; // Outputs: H
            echo $str[1]; // Outputs: e
            ?>

Arrays in PHP

Arrays are used to store multiple values in a single variable. PHP supports both indexed and associative arrays.

Introduction to Arrays

An array is a collection of values, and each value is identified by an index or a key. PHP provides a flexible way to work with arrays.

Creating Arrays

You can create arrays using the array() function or the short array syntax []: 

            <?php
            // Indexed array
            $fruits = array("Apple", "Banana", "Cherry");
            $vegetables = ["Carrot", "Pea", "Potato"]; // Short syntax
            ?>

Accessing Array Elements

You can access array elements using their index:

            <?php
            echo $fruits[1]; // Outputs: Banana
            ?>

2D Arrays

PHP supports multidimensional arrays, such as 2D arrays:

            <?php
            $students = [
                ["name" => "John", "age" => 20],
                ["name" => "Doe", "age" => 22]
            ];
            echo $students[0]["name"]; // Outputs: John
            ?>

Inspecting Arrays

You can use functions like print_r() and var_dump() to inspect array contents: 

            <?php
            print_r($fruits); // Outputs the structure of the array
            var_dump($fruits); // Provides more detailed information
            ?>

Deleting from an Array

You can delete an element from an array using the unset() function:

            <?php
            unset($fruits[1]); // Removes 'Banana'
            print_r($fruits); // Outputs: Array ( [0] => Apple [2] => Cherry )
            ?>

Iterating with each() and foreach()

The foreach() loop is the most common way to iterate over arrays:

            <?php
            foreach ($fruits as $fruit) {
                echo $fruit . "
"; // Outputs: Apple, Cherry
            }
            ?>

Note: each() is deprecated in PHP 7.2. It's better to use foreach().

Iterative Functions: current(), next(), prev(), reset(), end()

PHP provides several functions to navigate through an array:

            <?php
            $arr = ["a", "b", "c", "d"];
            
            echo current($arr); // Outputs: a
            next($arr);        // Moves pointer to the next element
            echo current($arr); // Outputs: b
            prev($arr);        // Moves pointer to the previous element
            echo current($arr); // Outputs: a
            reset($arr);       // Moves pointer back to the first element
            echo current($arr); // Outputs: a
            end($arr);         // Moves pointer to the last element
            echo current($arr); // Outputs: d
            ?>

Working with HTML Forms and PHP

HTML forms are used to collect user input, which can then be processed using PHP. This section covers the essential aspects of handling forms.

HTML Form Controls

Common form controls include:

  • <input>: For text, password, checkbox, radio buttons, etc.
  • <select>: Dropdown lists.
  • <textarea>: Multi-line text input.

Here’s an example of a simple form:

            <form action="process.php" method="post">
                Name: <input type="text" name="name"><br>
                Age: <input type="number" name="age"><br>
                Favorite Fruit: 
                <select name="fruit">
                    <option value="apple">Apple</option>
                    <option value="banana">Banana</option>
                    <option value="cherry">Cherry</option>
                </select><br>
                <input type="submit" value="Submit">
            </form>

Superglobal Variables

Superglobal variables are built-in global arrays in PHP that provide access to various data:

  • $_GET: For retrieving data sent via URL parameters.
  • $_POST: For retrieving data sent via form submissions.
  • $_REQUEST: Combines data from both $_GET and $_POST.
  • $_SESSION: For session variables.
  • $_COOKIE: For cookie variables.

Superglobal Arrays

Superglobal arrays allow easy access to user input and server information:

            <?php
            // Assuming the form is submitted
            if ($_SERVER["REQUEST_METHOD"] == "POST") {
                $name = $_POST["name"];
                $age = $_POST["age"];
                $fruit = $_POST["fruit"];
                echo "Name: $name, Age: $age, Favorite Fruit: $fruit";
            }
            ?>

Importing User Input

User input can be imported using the $_POST or $_GET superglobal arrays. For instance:

            <form action="" method="post">
                Name: <input type="text" name="name"><br>
                <input type="submit" value="Submit">
            </form>
    
            <?php
            if ($_SERVER["REQUEST_METHOD"] == "POST") {
                $name = htmlspecialchars($_POST["name"]); // Sanitizing input
                echo "Hello, $name!";
            }
            ?>

Accessing User Input

Access user input using the superglobal arrays. Always sanitize user input to prevent security vulnerabilities:

            <?php
            if ($_SERVER["REQUEST_METHOD"] == "POST") {
                $age = intval($_POST["age"]); // Convert to integer
                if ($age < 18) {
                    echo "You are a minor.";
                } else {
                    echo "You are an adult.";
                }
            }
            ?>

Database Connectivity in PHP

PHP provides various methods to connect and interact with databases. This section covers the essentials of working with MySQL databases.

Connecting to a Database

You can connect to a MySQL database using either MySQLi or PDO:

Using MySQLi

            <?php
            $conn = new mysqli("localhost", "username", "password", "database");
    
            // Check connection
            if ($conn->connect_error) {
                die("Connection failed: " . $conn->connect_error);
            }
            echo "Connected successfully"; // Outputs: Connected successfully
            ?>

Using PDO

            <?php
            try {
                $pdo = new PDO("mysql:host=localhost;dbname=database", "username", "password");
                $pdo->setAttribute(PDO::ATTR_ERRMODE, PDO::ERRMODE_EXCEPTION);
                echo "Connected successfully"; // Outputs: Connected successfully
            } catch (PDOException $e) {
                echo "Connection failed: " . $e->getMessage();
            }
            ?>

Making SQL Queries

You can create SQL queries to interact with the database:

            <?php
            $sql = "SELECT id, name FROM users";
            ?>

Executing Queries

Execute the SQL query using the connection:

            <?php
            // Using MySQLi
            $result = $conn->query($sql);
    
            // Using PDO
            $stmt = $pdo->query($sql);
            ?>

Fetching Data Sets

You can fetch results from a query:

            <?php
            // Using MySQLi
            if ($result->num_rows > 0) {
                while($row = $result->fetch_assoc()) {
                    echo "id: " . $row["id"]. " - Name: " . $row["name"]. "<br>";
                }
            } else {
                echo "0 results"; // Outputs: id: 1 - Name: John
 id: 2 - Name: Doe
            }
    
            // Using PDO
            while ($row = $stmt->fetch(PDO::FETCH_ASSOC)) {
                echo "id: " . $row["id"]. " - Name: " . $row["name"]. "<br>";
            }
            ?>

Integrating Forms and Databases

You can submit form data to a database:

Basic Form Submission to a Database

Working with Cookies and Sessions in PHP

Cookies and sessions are essential for managing state and user data in web applications. This section covers the basics of both.

Introduction to Cookies

Cookies are small pieces of data stored on the client’s browser that can be used to remember information about the user across sessions.

Setting Time in a Cookie with PHP

You can set a cookie using the setcookie() function. Here’s how to set a cookie with a time limit:

            <?php
            // Set a cookie that expires in 1 hour
            setcookie("username", "JohnDoe", time() + 3600);
            echo "Cookie 'username' is set!"; // Outputs: Cookie 'username' is set!
            ?>

Deleting a Cookie

To delete a cookie, set its expiration time to a time in the past:

            <?php
            // Delete the cookie
            setcookie("username", "", time() - 3600);
            echo "Cookie 'username' has been deleted!"; // Outputs: Cookie 'username' has been deleted!
            ?>

Creating a Session Cookie

A session cookie lasts only for the duration of the user's session. Here’s how to create one:

            <?php
            setcookie("session_cookie", "value", 0); // Expires when the browser closes
            echo "Session cookie is set!"; // Outputs: Session cookie is set!
            ?>

Introduction to Sessions

Sessions allow you to store data across multiple pages. Unlike cookies, session data is stored on the server.

Starting a Session

To use sessions, you need to start the session at the beginning of your script:

            <?php
            session_start();
            echo "Session started!"; // Outputs: Session started!
            ?>

Registering Session Variables

You can register session variables to store user data:

            <?php
            $_SESSION["username"] = "JohnDoe";
            echo "Session variable 'username' is set!"; // Outputs: Session variable 'username' is set!
            ?>

Working with Session Variables

To access session variables, simply reference them like this:

            <?php
            echo "Hello, " . $_SESSION["username"]; // Outputs: Hello, JohnDoe
            ?>

Destroying a Session

To destroy a session, use the following code:

            <?php
            session_start();
            session_destroy();
            echo "Session destroyed!"; // Outputs: Session destroyed!
            ?>

Passing Session IDs

Session IDs can be passed through URLs or forms. Here’s how to retrieve the session ID:

            <?php
            session_start();
            echo "Session ID: " . session_id(); // Outputs: Session ID: 
            ?>

Encoding and Decoding Session Variables

You can encode session variables using serialize() and decode them using unserialize(): 

            <?php
            $_SESSION["data"] = serialize(["item1", "item2"]);
            echo "Data stored in session: " . $_SESSION["data"]; // Outputs: Data stored in session: a:2:{i:0;s:5:"item1";i:1;s:5:"item2";}
            
            $data = unserialize($_SESSION["data"]);
            echo "First item: " . $data[0]; // Outputs: First item: item1
            ?>

Increasing Session Expire Time

You can increase the expiration time of a session like this:

            <?php
            session_start();
            // Set a new session expire time
            setcookie(session_name(), session_id(), time() + 3600);
            echo "Session expiration time extended!"; // Outputs: Session expiration time extended!
            ?>

Working of Session without Cookie

To work with sessions without cookies, you can pass the session ID in the URL:

            <?php
            session_start();
            // Check if session ID is passed in the URL
            if (isset($_GET['PHPSESSID'])) {
                session_id($_GET['PHPSESSID']);
            }
            echo "Session ID used: " . session_id(); // Outputs: Session ID used: 
            ?>

Working with the File System in PHP

PHP provides a variety of functions to work with files and directories, allowing you to read, write, and manage file permissions.

Understanding PHP File Permissions

File permissions determine who can read, write, or execute a file. In PHP, you can use chmod() to change file permissions: 

            <?php
            chmod("example.txt", 0755); // Sets read, write, and execute for owner, and read and execute for group and others
            echo "File permissions changed!"; // Outputs: File permissions changed!
            ?>

Opening and Closing a File

To work with files, you first need to open them using fopen(). Remember to close the file after using it with fclose():

            <?php
            // Open a file
            $file = fopen("example.txt", "r");
            if ($file) {
                echo "File opened successfully!"; // Outputs: File opened successfully
                fclose($file); // Close the file
            } else {
                echo "Failed to open file."; // Outputs: Failed to open file.
            }
            ?>

File Reading Functions

You can read files using various functions. Here are a few examples:

Reading Entire File

            <?php
            $content = file_get_contents("example.txt");
            echo "File Content: " . $content; // Outputs the content of the file
            ?>

Reading Line by Line

            <?php
            $file = fopen("example.txt", "r");
            while (($line = fgets($file)) !== false) {
                echo "Line: " . $line . "<br>"; // Outputs each line of the file
            }
            fclose($file);
            ?>

File Writing Functions

To write to a file, you can use the following functions:

Writing to a File

            <?php
            $file = fopen("example.txt", "w");
            fwrite($file, "Hello, World!"); // Write to the file
            fclose($file);
            echo "Data written to file!"; // Outputs: Data written to file!
            ?>

Appending to a File

            <?php
            $file = fopen("example.txt", "a");
            fwrite($file, "\nAppended text."); // Append to the file
            fclose($file);
            echo "Data appended to file!"; // Outputs: Data appended to file!
            ?>

File System and Directory Functions

PHP provides various functions to manage files and directories:

Creating a Directory

            <?php
            mkdir("new_directory"); // Creates a new directory
            echo "Directory created!"; // Outputs: Directory created!
            ?>

Checking if a File Exists

            <?php
            if (file_exists("example.txt")) {
                echo "File exists."; // Outputs: File exists.
            } else {
                echo "File does not exist."; // Outputs: File does not exist.
            }
            ?>

Deleting a File

            <?php
            if (unlink("example.txt")) {
                echo "File deleted!"; // Outputs: File deleted!
            } else {
                echo "Error deleting file."; // Outputs: Error deleting file.
            }
            ?>

Reading Directory Contents

            <?php
            $dir = "new_directory";
            if ($handle = opendir($dir)) {
                echo "Directory contents:
";
                while (false !== ($entry = readdir($handle))) {
                    echo "$entry
"; // Outputs the contents of the directory
                }
                closedir($handle);
            }
            ?>
